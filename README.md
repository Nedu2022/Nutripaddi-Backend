# Foodscan Backend

FastAPI backend for food image scanning and maternal nutrition advice.

## Required Hugging Face Secrets

Add these in the Space settings under **Variables and secrets**:

- `USDA_API_KEY`
- `HF_TOKEN`
- `GEMINI_KEY`
- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
- `AUTH_REQUIRED`
